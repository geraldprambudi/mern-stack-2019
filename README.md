# Mern-Stack-2019
(Mongo Express React Nodejs)

# Requirements
1. Node js and NPM
2. Mongodb
3. Postman
4. Code Editor(visual code, sublime text, atom, etc)

# How To Use
1. active your mongodb
2. install npm both server and client
```
// React js
$ cd /client 
$ npm install 
$ npm start
```

```
// Server Express js
$ cd /server
$ npm install
$ npm start
```

3. Open Your Browser
http://localhost:3000/index

That's it.
